Freeze Headers on Tree View module is a very useful Odoo addon for viewing tree view headers in Odoo.

Key Features

  • Headers in list views in Odoo can be viewed easily through this module.
  • Works on every tree view in Odoo
  • Useful when trying to deal with large no of records to keep track of there headers.
  • Supported by all browsers without any additional configuration.

Original Tree Views

Click on the Sales menu and open Product list view for a demo.

We can see that after scrolling down, the headers in the tree view are not visible in Odoo.

You can see that this happens to all the tree/list views in Odoo.


Freeze Header Module

After Installing the module, we can easily access the headers even after scrolling to the end.


Works Everywhere

The module works in all List views even in one2many and many2many list view.
